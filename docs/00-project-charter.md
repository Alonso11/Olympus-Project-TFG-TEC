---

## 2. Project Objectives

### 2.1 Primary Objective
Design and develop an autonomous Mars rover system capable of planetary surface exploration, scientific data collection, and autonomous navigation.

### 2.2 Secondary Objectives
- Implement autonomous obstacle avoidance
- Enable remote scientific data transmission
- Demonstrate fault-tolerant operation
- Validate system architecture through simulation

---

## 3. Scope

### 3.1 In Scope
- Autonomous navigation system
- Scientific instrumentation payload
- Communication subsystem
- Power management system
- Ground control station interface

### 3.2 Out of Scope
- Launch vehicle integration
- Deep space communication infrastructure
- Manufacturing and assembly

---

## 4. Constraints

### 4.1 Technical Constraints
- Limited computational resources (radiation-hardened processors)
- Communication delay (4-20 minutes one-way)
- Power limitations (solar panel efficiency)

### 4.2 Schedule Constraints
- Academic calendar timeline
- Milestone deliveries aligned with TEC requirements

### 4.3 Budget Constraints
- Academic project budget limitations
- Use of simulation/emulation environments
