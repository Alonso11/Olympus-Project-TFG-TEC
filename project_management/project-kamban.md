# Project Kanban - Olympus Project

Este archivo centraliza la gestion de tareas y el estado del desarrollo del proyecto.

## Enlace al Tablero de Gestion
El seguimiento detallado de las actividades se realiza en la plataforma de GitHub Projects:

[Acceder al Tablero de GitHub Projects](https://github.com/Alonso11/Olympus-Project-TFG-TEC/projects)

---

## Flujo de Trabajo (Workflow)
Se definen las siguientes etapas para el control de las unidades de trabajo:

1. Backlog: Requisitos y tareas identificadas pendientes de priorizacion.
2. To Do: Tareas seleccionadas para ejecucion inmediata.
3. In Progress: Actividades en fase de desarrollo activo.
4. In Review: Tareas finalizadas en proceso de verificacion o revision tecnica.
5. Done: Tareas completadas y validadas.

---

## Directrices de Gestion
- El registro de tareas debe realizarse preferiblemente mediante Issues de GitHub.
- Todo cambio significativo en el codigo o documentacion debe estar referenciado a su respectiva tarea.
