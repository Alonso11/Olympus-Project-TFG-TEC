\section{Integración del Prototipo (Objetivo Específico 3)}

\subsection{Estrategia de Integración}
Se seguirá un enfoque de \textbf{integración incremental} y por capas, minimizando riesgos en etapas tempranas.

\begin{itemize}
    \item \textbf{Fase 1: Integración de Software (In-Silico):} Pruebas en simuladores (Gazebo/Isaac Sim) para validar algoritmos de navegación.
    \item \textbf{Fase 2: Integración de Hardware Base:} Ensamblaje del chasis, motores y sistema de potencia básica.
    \item \textbf{Fase 3: Integración de Electrónica de Control:} Conexión de microcontroladores y sensores de bajo nivel.
    \item \textbf{Fase 4: Integración de Sistema Completo:} Unión de las capas de software de alto nivel con el hardware físico.
\end{itemize}

\subsection{Configuración del Prototipo}
\begin{enumerate}
    \item \textbf{Versión 1.0 (MVP):} Prototipo con movilidad básica controlada remotamente.
    \item \textbf{Versión 2.0 (Autónomo):} Integración de LiDAR y navegación autónoma en entorno controlado.
    \item \textbf{Versión 3.0 (Final):} Sistema integrado con gestión energética y sensores científicos para ensayos en entorno análogo.
\end{enumerate}
