\section{Arquitectura del Sistema}

\subsection{Arquitectura Funcional y Flujo de Datos}
El Rover Olympus se basa en un modelo de control de bucle cerrado dividido en bloques funcionales que abstraen el hardware de la lógica de misión:

\begin{itemize}
    \item \textbf{Bloque de Percepción:} Transforma datos crudos (LiDAR, IMU, Encoders) en un modelo del entorno (mapas de ocupación) y un estado de navegación (pose).
    \item \textbf{Bloque de Decisión y Gestión de Misión:} Implementa una Máquina de Estados Finitos (FSM) que orquesta los modos de operación (Exploración, Seguro, Retorno) según los comandos de la GCS y el estado del sistema.
    \item \textbf{Bloque de Control y Actuación:} Genera comandos de velocidad (Twist) y los traduce a señales de bajo nivel para los controladores de motores mediante el protocolo CSP.
\end{itemize}

\subsection{Arquitectura de Comunicaciones: Stack de Protocolos}
El sistema utiliza una arquitectura orientada a servicios basada en el \textit{CubeSat Space Protocol} (CSP), permitiendo una comunicación transparente entre subsistemas heterogéneos:

\begin{itemize}
    \item \textbf{Capa de Aplicación:} Tópicos ROS 2 y Servicios de Gestión de Archivos.
    \item \textbf{Capa de Transporte:} CSP (Protocolo de red espacial con direccionamiento de nodos y puertos).
    \item \textbf{Capa de Enlace:} 
        \begin{itemize}
            \item \textbf{Interna:} CAN Bus (Motores) e I2C (Sensores).
            \item \textbf{Externa (Tierra):} Enlace de radiofrecuencia (RF) con encapsulación CSP.
        \end{itemize}
\end{itemize}

\subsection{Arquitectura de Software (ROS 2)}
La implementación de software se divide en capas de abstracción para asegurar la mantenibilidad (RNF-MTN-01):
\begin{enumerate}
    \item \textbf{Hardware Abstraction Layer (HAL):} Nodos ROS que interactúan con el hardware mediante \textit{drivers} específicos (libcsp, rclcpp).
    \item \textbf{Capa Middleware:} Sistema de transporte basado en DDS (Data Distribution Service) configurado para entornos de baja latencia.
    \item \textbf{Capa de Aplicación:} Algoritmos de SLAM, Nav2 y el orquestador de misión.
\end{enumerate}
